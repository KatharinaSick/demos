# Demos — Agent Context

This repo contains self-contained conference/talk demos. Each demo lives in its own subdirectory with its own devcontainer, cluster manifests, and application code.

For demo-specific architecture, known gotchas, and design decisions, see the `AGENTS.md` in each subdirectory.

## Keeping these files up to date

If you learn something worth remembering during a conversation — a non-obvious design decision, a fixed bug, a gotcha, a changed credential or version — add it to the relevant `AGENTS.md` before the session ends.
